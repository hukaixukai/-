# cli/ — Rich 交互式命令行前端

本模块提供了基于 Rich 库的美化命令行界面，是当前的主要用户交互入口。

## 文件说明

| 文件 | 用途 |
|---|---|
| `interface.py` | CLI 主界面：菜单、对话、知识库管理、练习等全部交互逻辑 |

## 功能菜单

启动后进入主菜单，提供以下功能：

1. **智能问答** — 进入对话模式，支持流式输出、RAG 自动检索、Markdown 渲染
2. **知识库管理** — 导入文件/目录、查看已导入资料、清空知识库
3. **学情诊断** — 展示知识点掌握情况表格、薄弱/强势分析、可选 AI 深度诊断
4. **练习与测验** — 选择题/填空题/简答题/编程题，交互式答题+即时反馈
5. **学习计划** — 基于学情自动生成个性化学习计划
6. **切换课程** — 多学科切换，各自独立历史和知识库
7. **学生设置** — 配置学生 ID，支持多学生使用

## 特性

- 流式对话输出（逐字显示 AI 回复）
- RAG 自动检测：知识库有内容时自动检索参考资料
- 答题结果自动记录到学生档案
- 支持 `/quit` 退出对话、`/clear` 清空历史
- Rich Markdown 渲染、进度条、表格、面板
